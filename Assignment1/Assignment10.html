HTML COMMENTS AND SPECIAL CHARACTERS

*COMMENTS IN HTML*

HTML, comments are written using "". They allow you to add notes or explanations within the code that are not displayed in the browser.

Comments in HTML are not displayed in the browser but can help in explaining the code.

*SPECIAL CHARACTERS*

HTML character entities are basically a set of characters (entity) used to represent few characters reserved by the HTML, especially invisible characters or characters difficult to type out using a regular keyboard. HTML provides some entity names and entity numbers to use these symbols.

Here are some special charcters displayed using HTML entities:

  • Ampersand: &
  • Less Than: <
  • Greater Than: >
  • Copyright: ©
  • Registered Trademark: ®
  • Trademark: ™
